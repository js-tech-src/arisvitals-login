Aris - Politica de PrivacidadeDocumento de referencia para a fase de prova de conceito (PoC). Sera substituido por politica formal antes de qualquer lancamento publico.

O que e o Aris

Aris e um assistente pessoal de saude que ajuda o usuario a registrar e acompanhar eventos de bem-estar (peso, alimentacao, agua, exercicio, suplementacao).

Dados coletados

Eventos de saude relatados pelo proprio usuario (ex.: peso, refeicoes, agua, exercicios, suplementos) e o e-mail usado para autenticacao.

Como os dados sao usados

Exclusivamente para registrar e devolver ao proprio usuario seu historico de saude, e para gerar orientacao baseada nesse historico. Dados nao sao compartilhados com terceiros nem usados para publicidade.

Isolamento entre usuarios

Cada usuario so tem acesso aos proprios dados, garantido por controle de acesso a nivel de banco de dados (Row Level Security), nao apenas por logica de aplicacao.

Autenticacao

O login e feito via e-mail e senha, processado pelo Supabase Auth. A senha nao e armazenada nem visivel para o assistente em nenhum momento.
